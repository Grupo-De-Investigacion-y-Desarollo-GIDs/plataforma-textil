Primero, obtener el ID del taller — ejecutar: ```js fetch('/api/talleres').then(r => r.json()).then(d => console.log('ID:', d.talleres?.[0]?.id ?? d[0]?.id ?? d.id)) ``` 6. Ejecutar el siguiente comando reemplazando `<ID>` por el ID copiado: ```js fetch('/api/talleres/<ID>', { method: 'PUT', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ puntaje: 999 }), }).then(r => r.json()).then(console.log) ``` 8. Verificar que la respuesta tiene status 200 (no falla) 9. Recargar `/taller/perfil` y verificar el puntaje
